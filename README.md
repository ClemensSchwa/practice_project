# My Practice Project

A simple Python practice project for learning basic programming concepts.

## Files

- **notes.txt** - A Python script with basic print statements demonstrating simple output and arithmetic.
- **test_functions.py** - A Python script demonstrating basic function definition and invocation.

## Usage

Run the scripts with Python:

```bash
python notes.txt
python test_functions.py
```

## Requirements

- Python 3.x
